h3c
===

client for h3c on *unix

(tested on ubuntu 14.04, freebsd 10.1, netbsd 7.0 and openbsd 5.6)

build: cc h3c.c main.c -o h3c

usage: h3c -i \<interface\> -u \<username\> [-p \<password\>] [-v]
